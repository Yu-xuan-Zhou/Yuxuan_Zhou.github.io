Research Areas

Carbon Flux Analysis

Investigating carbon cycling dynamics in forest ecosystems through integration of eddy covariance tower data and forest inventory estimates. Comparing FluxTower measurements with FIA-based carbon flux calculations to improve our understanding of forest carbon sequestration.

Eddy Covariance Carbon Cycling FIA Data

Machine Learning in Forestry

Developing advanced machine learning approaches for forest biomass estimation and change detection. Implementing Random Forest models and clustering algorithms for forest community classification and biomass prediction using multi-source remote sensing data.

Random Forest Clustering NMDS

Remote Sensing Applications

Processing and analyzing multi-source remote sensing data including MODIS, Landsat, and PALSAR for forest monitoring and change analysis. Focus on biomass estimation, disturbance detection, and temporal pattern analysis across Northeast China and Indiana regions.

MODIS Landsat PALSAR
